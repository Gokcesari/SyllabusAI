COURSE & SYLLABUS

First create the course (code + title). Then upload your syllabus file — e.g. COP4803 SYLLABUS.docx as Word or PDF. Students can only enroll with a course code that exists in the system.

Step 1 — Course details

Students enter this code on the Add course screen; they cannot enroll with a code that does not exist.

Optional: paste text manually (if you will not use a file)

Step 2 — Syllabus file (PDF or Word)

Your syllabus is usually Word or PDF; both are supported. Text is extracted and stored for search and AI chunks.

Example: COP4803 SYLLABUS.docx or PDF syllabus. Max ~20 MB.
